# Movie API

## Description

### This is a simple movie api, with basic curd operations with Golang as Backend and Mongodb as Primary DataBase

## Installation

### 1. Go Mod Download

```
go mod download
```

#### Folder Structure

```
- controllers
    - controller.go
- models
    - models.go
- routers
    - routers.go
- go.mod
- main.go
```
